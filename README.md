# about

simple audio flac player for smallest system

1. audio output - alsa or pulse
2. flac - 192000, 96000, 48000, 44100 Hz, 16 or 24 bit stereo format

# requirement

1. g++ (version 12.2.0 or newest)
2. make
3. libasound2-dev
4. libpulse-dev
5. libflac++-dev

# build

make

# install

make install
