# about

audio flac player for smallest system

1. ui - ncurses
2. audio output - alsa
3. flac - 192000, 96000, 48000, 44100 Hz, 16 or 24 bit stereo format

# requirement

1. clang-20
2. make
3. libasound2-dev
4. libflac++-dev
5. libncurses5-dev

# build

make

# install

make install

# control

1. up - up in  music list
2. down - down music list
3. left - push dir
4. right - pop dir
5. q - quit
6. p - play
7. s - stop
